# DataTello

Evidence-backed **decision engine** for build opportunities — tells each user what to build or act on first.

Discovers overlooked environmental compliance, contractor safety, and public-sector workflow problems — validates them through layered evidence — and recommends your next move by role.

**Not** a research dashboard or general market intelligence platform.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001)
